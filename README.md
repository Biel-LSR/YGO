## Yu-gi-oh! Card Viewer
Bem-vindo ao Yu-gi-oh! Card Viewer! Este projeto apresenta minhas habilidades e projetos como Desenvolvedor Web Designer UI/UX.

### 🚀 Sobre o Projeto
Esta é uma página para portfólio pessoal desenvolvida para showcase de trabalhos e competências. O objetivo é fornecer uma visão clara e moderna sobre meus projetos e gostos, facilitando o contato com recrutadores e clientes.

 ### 🎯 Funcionalidades
Carrossel dinâmico com temporizador para troca de cards
Design Responsivo: Layout adaptável para dispositivos móveis, tablets e desktops.

### 💻 Tecnologias Utilizadas
Frontend: HTML5, CSS3, JavaScript (ES6+)
Estilização: CSS3
Ferramentas: Git, GitHub, VScode
Hospedagem: Cloudflare

Este projeto está configurado para deploy automático na GitHub Pages. A cada push para a branch main, o site é automaticamente atualizado.
